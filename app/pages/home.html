⚙️ Tecnologia
Como o SkyGuard Funciona

Arquitetura simples, robusta e escalável para ambientes acadêmicos, industriais e comerciais.

🔬
Sensor SGP30
O SGP30 é um sensor de gás digital que mede CO₂ equivalente (eCO₂) e TVOC com alta precisão. Utiliza tecnologia MOX para detecção química.
📶
ESP32 + Wi-Fi
O microcontrolador ESP32 coleta os dados do sensor e os publica via protocolo MQTT em um broker, enviando leituras a cada 4 segundos.
☁️
MQTT Broker
Protocolo leve ideal para IoT. Os dados são publicados em tópicos como skyguard/SGP-001/data e recebidos pelo servidor em tempo real.
🗄️
PHP + MySQL
O backend em PHP recebe os dados MQTT, os armazena no banco MySQL (phpMyAdmin) e os disponibiliza para a interface web via API REST.
📊
Dashboard Web
Interface web responsiva que exibe as leituras em tempo real, histórico de medições, alertas e status dos dispositivos cadastrados.
🔔
Alertas Inteligentes
Notificações automáticas quando CO₂ ou TVOC ultrapassam limites configurados, auxiliando na tomada de decisão imediata.

🏢 Sobre a Empresa
SkyGuard

A SkyGuard nasceu da necessidade de tornar o monitoramento da qualidade do ar acessível, inteligente e em tempo real. Nossa plataforma integra hardware de baixo custo com software profissional para oferecer soluções completas para escolas, laboratórios e empresas.

Desenvolvido como projeto de Trabalho de Conclusão de Curso (TCC), o sistema demonstra como tecnologias IoT podem ser aplicadas para resolver problemas reais de saúde e bem-estar em ambientes fechados.
